Wild Physics

Inicio:         * Comenzamos con 500 puntos.
Mecanica:       * Sumamos puntos acertando el disparo a las bolas de colores (cada una con distinto puntaje).
                * Restamos puntos si las bolas de colores tocan el piso (cada una con distinto puntaje)
                * Ganamos cuando no quedan bolas en pantalla.
Objetivo:       * Acertar el disparo a todas las bolas de colores y terminar con el puntaje mas alto posible.
Salir:          * Presionar la tecla "Escape" cuando finaliza el juego para cerrar la ventana.

¡¡Diviertete!!!
